# 🐝 Beecrowd-JavaScript

Esse repositório contém minhas **soluções em JavaScript** para os problemas da plataforma **Beecrowd (antigo URI Online Judge)**.  
Exploro lógica de programação e resolução de desafios com foco em entradas e saídas no padrão da Beecrowd.

## 📌 Sobre o projeto

Beecrowd é uma plataforma de competições e desafios de programação composta por centenas de problemas em diferentes níveis de dificuldade. Neste repositório, eu resolvo cada desafio usando **JavaScript**, normalmente com Node.js. :contentReference[oaicite:2]{index=2}

Aqui você encontra:

✔️ Soluções dos problemas iniciais  
✔️ Exercícios de fluxo de controle  
✔️ Lógica matemática  
✔️ Manipulação de entrada e saída padrão  

Cada arquivo `.js` corresponde a um problema da plataforma e é nomeado com o **número do problema**.
